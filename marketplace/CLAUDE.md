# Marketplace Directory

Plugin marketplace metadata and example outputs for PMCopilot distribution.

## Contents

| File/Directory | Purpose |
|----------------|---------|
| `listing.md` | Marketplace listing copy — description, capabilities, integrations, and getting started instructions |
| `examples/` | Example outputs showcasing what each command produces |

## Example Outputs

The `examples/` directory contains sample artifacts generated by PMCopilot commands:

- `example-prd-output.md` — Sample PRD from the prd command
- `example-prioritization-output.md` — Sample RICE/ICE scoring output
- `example-teardown-output.md` — Sample competitive teardown report

These are used in the marketplace listing and README to demonstrate PMCopilot's capabilities.

## Maintenance

- Update `listing.md` when commands, agents, or integrations change
- Regenerate example outputs when command output format changes significantly
