## Description

Create the persona template at `skills/user-research/templates/persona.md`. This template provides a structured format for documenting user personas derived from research. Personas help the team build empathy and make user-centered decisions by representing key user archetypes.

The template must include all standard persona fields to paint a complete picture of the user archetype.

## Acceptance Criteria

- [ ] `skills/user-research/templates/persona.md` exists
- [ ] Template includes a name field (archetypal name)
- [ ] Template includes a photo placeholder/description field
- [ ] Template includes a role/job title field
- [ ] Template includes an age field
- [ ] Template includes a location field
- [ ] Template includes a tech savviness field (e.g., scale or descriptor)
- [ ] Template includes a goals section (what the persona is trying to achieve)
- [ ] Template includes a frustrations section (current pain points)
- [ ] Template includes a representative quote field
- [ ] Template includes a day-in-the-life narrative section
- [ ] Template includes a current solutions section (tools/workarounds they use today)
