## Description

Create the interview guide template at `skills/user-research/templates/interview-guide.md`. This template provides a structured script for conducting user interviews. It walks the interviewer through a logical flow from screening and warm-up through core questioning to wrap-up, ensuring consistent and thorough data collection across interviews.

## Acceptance Criteria

- [ ] `skills/user-research/templates/interview-guide.md` exists
- [ ] Template includes a research objective section (what you want to learn and why)
- [ ] Template includes a screening section (criteria for participant eligibility and screening questions)
- [ ] Template includes a warm-up section (introductory questions to build rapport and set context)
- [ ] Template includes a core questions section organized by topic area, with open-ended questions
- [ ] Template includes follow-up probes for each topic area (deeper-dive prompts to elicit richer responses)
- [ ] Template includes a wrap-up section (closing questions, opportunity for participant to add anything, next steps)
- [ ] Template includes practical guidance (estimated time per section, tips for active listening, note-taking reminders)
