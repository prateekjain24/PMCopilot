## Description

Create the JTBD canvas template at `skills/user-research/templates/jtbd-canvas.md`. This template provides a structured canvas for mapping out Jobs to Be Done, following the Outcome-Driven Innovation framework. It helps PMs decompose a user&apos;s core job into its constituent parts and identify underserved needs and innovation opportunities.

## Acceptance Criteria

- [ ] `skills/user-research/templates/jtbd-canvas.md` exists
- [ ] Template includes a job performer field (who is performing the job)
- [ ] Template includes a core functional job field (the main job the user is trying to get done)
- [ ] Template includes a related jobs section (adjacent jobs the performer also needs to do)
- [ ] Template includes an emotional jobs section (how the performer wants to feel)
- [ ] Template includes a social jobs section (how the performer wants to be perceived)
- [ ] Template includes an 8-step job map with all steps: Define, Locate, Prepare, Confirm, Execute, Monitor, Modify, Conclude
- [ ] Each step in the job map includes a field for pain points and unmet needs
- [ ] Template includes a desired outcomes section for capturing measurable outcome statements
